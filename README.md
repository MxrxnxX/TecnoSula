# TecnoSula

Sistema de gestión de quejas y tickets.

## Tecnologías

Backend:
- ASP.NET Core .NET 10
- Entity Framework Core
- SQL Server
- JWT
- BCrypt

## Funcionalidades actuales

- Registro de usuarios
- Login con JWT
- Autenticación por roles
- Gestión de usuarios

## Próximamente

- Módulo de tickets
- Dashboard
- Reportes
